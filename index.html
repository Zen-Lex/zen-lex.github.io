Exercice 4 : nombre de clics

La page HTML initiale doit contenir un paragraphe contenant le chiffre 0 : <p>0</p>.
Écrire une fonction JS qui incrémente ce chiffre de 1 à chaque fois que l'on clique sur le paragraphe.
Note : Une actualisation de la page remet bien entendu le compteur à 0.

Code:

function Click() {
	let nbClick = 0;
	let p = document.createElement("p");
		
	document.body.section.appendChild(p).append(nbClick);
	p.addEventListener("click", function(){
		nbClick++;
		p.innerText = nbClick;
	});
}
